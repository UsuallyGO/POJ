USACO MAR07 Problem 'latin' Analysis

by Rob Kolstad

This is a straightforward program if the input and output can be handled.

Once the word is read, the first character is examined and an appropriate print statement is selected.

The JAVA solution of USA's Brian O'Connor follows: 

import java.io.*;
import java.util.*;

class latin {
  public static void main (String [] args) throws IOException {
    BufferedReader f = new BufferedReader(new FileReader("latin.in"));
    PrintWriter out = new PrintWriter(new BufferedWriter(new FileWriter("latin.out")));

    int n = Integer.parseInt(f.readLine());

    for(int i = 0; i &lt; n; i++){
        String word = f.readLine();

        char first = word.charAt(0);

        if(first == 'a' || first == 'e' || first == 'i' || first == 'o' || first == 'u'){
            out.println(word + "cow");
        } else {
            out.println(word.substring(1) + first + "ow");
        }
    }
    out.close();
    System.exit(0);
  }
}
