USACO MAR07 Problem 'round' Analysis

by Bruce Merry

There are few enough possibilities that we can try them all and just see which satisfy the constraints. By fixing the position of one cow, we eliminate the issue of counting the same arrangement multiple times, and at the same time reduce the number of options to check by a factor of N.

The C++ Standard Template Library has a handy function called next_permutation that takes care of all the work in looping over the possible permutations.
